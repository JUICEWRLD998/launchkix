## 📋 Business Questionnaire

### 1. What does your application/service do?

LauchKix is an AI-powered platform that generates complete launch marketing kits for mobile apps in under 60 seconds. It transforms basic app details (name, description, target audience, tone) into 8 professional marketing sections:

- **App Store & Google Play listings** with character-optimized titles and descriptions
- **ASO keyword packs** with primary, secondary, and long-tail keywords plus optimization strategies
- **7-day social media calendar** with platform-specific posts for X, LinkedIn, and Instagram
- **Launch email campaigns** with multiple subject line variations
- **Community launch posts** formatted for Reddit, Product Hunt, and Indie Hackers
- **Landing page copy** with headlines, features, and CTAs
- **Press & media blurbs** with newswire summaries and founder quote templates
- **Multi-format export** in Markdown, JSON, or copyable text

The platform uses Gemini 2.0 Flash via OpenRouter with structured JSON output, tone customization (Professional, Playful, Bold, Friendly), and goal-focused generation (Downloads, Waitlist, Paid Conversions). Users can regenerate individual sections without losing other content, enabling iterative refinement.

## 💻 Quick Start

```bash
# Clone and install
git clone https://github.com/JUICEWRLD998/lauchkix.git
cd lauchkix
npm install

# Configure API key
cp .env.example .env.local
# Add your OPENROUTER_API_KEY to .env.local

# Run locally
npm run dev
# Open http://localhost:3000
```

**Usage:** Navigate to `/generate` → Fill in app details → Click "Generate Launch Kit" → Copy, regenerate, or export sections
